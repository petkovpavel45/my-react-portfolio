# Pavel Petkov — Portfolio

Personal developer portfolio built with React and Tailwind CSS.

**[Live Demo →](https://portfolio-project-mgnx2gsvw-vibe-coding4.vercel.app)**

---

## Sections

- **Home** — intro with call-to-action buttons
- **About** — background, current role, contact info
- **Portfolio** — project cards with live demo and source links
- **Experience** — tech stack with skill icons
- **Contact** — contact form via getform.io
